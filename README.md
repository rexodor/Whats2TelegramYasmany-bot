# Whats2TelegramYasmany-bot
Un bot de Telegram sencillo que convierte el formato de texto de WhatsApp (*negrita* con `*` y *cursiva* con `_`) al formato nativo de Telegram (**negrita** con `**` y **cursiva** con `__`). Ideal para quienes quieren compartir mensajes de WhatsApp en chats de Telegram sin perder el estilo.
